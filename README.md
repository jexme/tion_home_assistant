# Tion Breezer & MagicAir (Async) for Home Assistant

Асинхронная интеграция для управления бризерами **Tion O₂ MAC 1** и датчиками **MagicAir** через Tion Cloud API.

## Особенности
* **Асинхронный движок (`aiohttp`)** — не блокирует Home Assistant.
* **Мгновенный отклик кнопок (Optimistic UI)** — устраняет визуальные задержки при переключении скоростей и режимов.
* **Авторизация через интерфейс** — настройка логина/пароля в веб-интерфейсе без YAML.

## Поддерживаемые функции

### Управление бризером
* **Термостат (`climate.tion_breezer_<guid>`)**: включение/выключение, обогрев (`heat` / `maintenance`), целевая температура.
* **Скорость вентилятора (`number.tion_breezer_speed_<guid>`)**: ручное управление скоростью бризера (от 0 до 6). Значение `0` полностью выключает устройство.

### Управление зоной MagicAir
* **Режим управления (`select.tion_zone_mode_<guid>`)**: переключение между ручным (`manual`) и автоматическим (`auto`) режимами управления зоной.
* **Целевой CO₂ (`number.tion_zone_co2_<guid>`)**: установка желаемого уровня CO₂ для автоматического режима (в ppm).

### Мониторинг (Сенсоры)
* **Показания MagicAir (`sensor.*`)**: текущий уровень CO₂, температура и влажность в помещении.
* **Показания бризера (`sensor.*`)**: температура притока/выхода воздуха, износ фильтра (в днях).

## Установка через HACS

1. Откройте **HACS** -> **Интеграции**.
2. В правом верхнем углу нажмите три точки -> **Пользовательские репозитории** (Custom repositories).
3. Добавьте URL репозитория: `https://github.com/jexme/tion_home_assistant` с категорией **Интеграция** (Integration).
4. Найдите **Tion Breezer & MagicAir (Async)** в списке, нажмите **Скачать** (Download).
5. **Перезагрузите Home Assistant**.

---

## Настройка

1. Перейдите в **Настройки** -> **Устройства и службы** (Settings -> Devices & Services).
2. Нажмите **Добавить интеграцию** и выберите **Tion Breezer & MagicAir (Async)**.
3. Введите **Email** и **Пароль** от MagicAir.
